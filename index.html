Anyone who is looking for software engineering job opportunities faces the daunting coding interview involving data structures and algorithms. Despite the multiple CS resources, students often find it difficult to understand advanced topics such as Graphs, Trees, Binary Heap, and Linked List with plain text.

Our group decided to introduce a visualization app on data structures and algorithm questions on these four subjects. This website serves as a visualization to these subjects with tech stack including HTML, CSS, Javascript and D3.js, which is a javascript library for producing dynamic, interactive data visualizations in web browsers.

To help the user understand in a clear manner, we will demonstrate visualization of Graphs, Trees, Binary Heap, and Linked List step by step. The website serves as an educational tool for those who are preparing coding interviews or who are looking to strengthen their Computer Science knowledge.
